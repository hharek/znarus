znarus
======
Smart and fast CMS (Content management system)

REQUIREMENTS:
- Apache (mod-php) or Nginx (fast-cgi php)
- PHP 5.4 and up
- PHP-module: pgsql, ftp, zip, mbstring
- PostgreSQL 9.0 and up
