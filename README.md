znarus
======
Smart and fast CMS (Content management system)

REQUIREMENTS:
- Nginx (fast-cgi php)
- PHP 7
- PHP-module: pgsql, zip, mbstring, curl, openssl
- PostgreSQL 9.4 and up

OPTIONAL:
- Sphinx Search
- Memcached
