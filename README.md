znarus
======
Smart and fast CMS (Content management system)

REQUIREMENTS:
- Nginx (fast-cgi php)
- PHP 5.6 and up
- PHP-module: pgsql, ftp, zip, mbstring, curl, openssl, db4
- PostgreSQL 9.3 and up

OPTIONAL:
- Sphinx Search
- Memcached
